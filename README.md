# nodejs-native-test-runner

## **Requirements**

Install Node.js 20.5.0 version:

```sh
node -v
# v20.5.0
```

Install dependencies:

```sh
npm ci
```

## **Running**

To start api:

```sh
npm start
```

Run tests:

```sh
npm t
```

Run tests in watch mode:

```sh
npm run test:watch
```

Run coverage:

```sh
npm run test:cov
```
